# Album Library App

The **Album Library** app is a mobile application that allows users to manage and explore their personal album collections. Users can view a list of albums, check detailed information about each album, and add or edit albums in their library. The app also supports real-time updates from the server to keep the library always in sync.

---

## Features

- Browse a list of albums in your personal library
- View detailed information for each album, including:
    - Title
    - Artist
    - Released Year Year
    - Genre
    - Track Count
    - Cover URL
    - Added At Date
- Add new albums to your library
- Edit existing album details
- Delete albums from your collection
- Receive real-time updates when albums are added, updated, or removed on the server
- Seamless navigation between album list (master) and album detail views
